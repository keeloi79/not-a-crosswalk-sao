# not-a-crosswalk-sao

# "Not Just a Crosswalk" SAO project for DEFCON 34

A hardware and firmware project for the DEFCON badge. This repository contains documentation, schematics, firmware, and build instructions.

## Overview

This badge is designed for DEFCON attendees to explore embedded systems, hardware hacking, and firmware development. It includes a microcontroller, display, buttons, and wireless connectivity.
