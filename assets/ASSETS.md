assets go here# Assets

This document serves as the central repository for all visual and technical assets related to the **Not Just a Crosswalk SAO** project. All files are hosted in the `assets/` directory and are intended for public use, documentation, and community reference.

## 📸 Visual Documentation

### Screenshots
- **`assets/screenshots/`**
  - `startup_rainbow_splash.png`: Boot animation showing the rainbow splash before user settings load.
  - `palette_cycle_demo.png`: Demonstration of the 13-color palette cycling through community flags.
  - `effect_demo.png`: Sample animation effects (e.g., "party_bright", "static", "pulse").
  - `button_press_demo.png`: Visual of the button interface and real-time state changes.

### High-Resolution PCB Images
- **`assets/pcb/`**
  - `Crosswalk_SA0_v8_PCB_Layout.pdf`: Full 2D layout of the SAO board (top and bottom layers).
  - `Crosswalk_SA0_v8_PCB_3D_Render.png`: 3D render of the badge with LED array and connector placement.
  - `Crosswalk_SA0_v8_PCB_Solder_Joints.pdf`: Close-up of critical solder points and component placement.

### Physical Photos
- **`assets/photos/`**
  - `badge_assembled.jpg`: Final assembled badge with LEDs off.
  - `badge_on_table.jpg`: Badge on a conference table with ambient lighting.
  - `badge_button_press.jpg`: Close-up of a user pressing the palette button.

### Schematics
- **`assets/schematics/`**
  - `Crosswalk_SA0_v8_Schematic.pdf`: Full schematic in PDF format, including power, logic, and interface circuits.
  - `Crosswalk_SA0_v8_Schematic_Simplified.pdf`: Condensed version for quick reference.
